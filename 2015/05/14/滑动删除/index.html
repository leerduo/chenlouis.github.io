完成SwipeView滑动的逻辑

SwipeView的滑动逻辑主要借助ViewDragHelper完成。ViewDragHelper封装了对触摸位置、触摸速度、触摸距离的检测，以及Scroller.在自定义ViewGroup的时候，他可以起到很大的作用。他提供了一系列的工具方法来追踪用户的拖拽事件以及重新定位view在ViewGroup中的位置。
在init()方法中创建它(通过静态工厂)

创建call接口，并实现几个需要的方法，由于我们这里是水平滑动，因此我们需要的方法都是很水平滑动相关的。

  • public boolean tryCaptureView(View child, int pointerId);
  • public void onViewCaptured(View capturedChild, int activePointerId);
  • public int getViewHorizontalDragRange(View child);
  • public int clampViewPositionHorizontal(View child, int left, int dx);
  • public void onViewPositionChanged(View changedView, int left, int top, int dx, int dy);
  • public void onViewReleased(View releasedChild, float xvel, float yvel);
